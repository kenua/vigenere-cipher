Vigenère cipher

The Vigenère cipher is a method of encrypting alphabetic text by using a series of interwoven Caesar ciphers, based on the letters of a keyword.

- Wikipedia

This this a webpage where you can encrypt or decrypt a message using the Vigenère cipher. To encrypt your message you need to write a Key.The Key field is used for encrypting your message, and It should be a single line of text (like dog, Maria, or helloWord) but it should not have any whitespace, punctuation mark, or weird character.

To decrypt your message you need to write the same Key used for encrypting it. The Key is case-insensitive, meaning the case of your key doesn't change the encryption and decryption result.
